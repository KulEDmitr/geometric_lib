# About geometric_lib
Using geometric formulas for a circle, rectangle, square and triangle, the area and perimeter are calculated.

# Math formulas
## Area
- Circle: S = πR²
> r = 3
> 
> area = 28.274333882308138
- Rectangle: S = ab
> a = 3; b = 4
>
> area = 12
- Square: S = a²
> a = 3
>
> area = 9
- Triangle: S = ah / 2
> a = 3; h = 6
>
> area = 9

## Perimeter

- Square: P = 4a
