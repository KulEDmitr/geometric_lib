# Math formulas
## Area
- Circle: S = πR²
```
area(10)=314
```
- Rectangle: S = ab
```
area(10, 20) = 200
```
- Square: S = a²
```
area(10) = 100
```
- Triangle: S = ah/2
```
area(10, 5) = 25
```

## Perimeter
- Circle: P = 2πR
```
perimeter(10) = 62,8
```
- Rectangle: P = 2a + 2b
```
perimeter(10, 20) = 60
```
- Square: P = 4a
```
perimeter(10) = 40
```
- Triangle: P = a + b + c
```
perimeter(10, 20, 5) = 35
```
