# General description of the solution
For each figure (from: **_circle, rectangle, square, triangle_**), 
functions for calculating **area** and **perimeter** have been created
## Math formulas
### Area:
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle S = (a * h) / 2

### Perimeter:
- Circle: P = 2πR
- Rectangle: P = 2(a + b)
- Square: P = 4a
- Triangle P = a + b + c
