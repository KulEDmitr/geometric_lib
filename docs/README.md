# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a


# Геометрические Вычисления

## circle.py


#### `area(r)`

Вычисляет площадь круга по заданному радиусу.

- **Параметры:**
  - `r` (float): Радиус круга.

- **Возвращает:**
  - `float`: Площадь круга.

- **Пример использования:**

```python
print(area(5))
# Вывод: 78.53981633974483
```


#### `perimeter(r)`

Вычисляет периметр (длину окружности) круга по заданному радиусу.

**Параметры:**

- `r` (float): Радиус круга.

**Возвращает:**

- `float`: Периметр окружности.

**Пример использования:**

```python
# Пример 1
radius = 5
print(perimeter(radius))
# Вывод: 31.41592653589793

# Пример 2
radius = 2.3
print(perimeter(radius))
# Вывод: 14.451350773405933
```


## square.py

### area(a)
вычисляет площадь квадрата с длиной стороны a.

Параметры:

- a (число): Длина стороны квадрата.

Возвращает:

- (число) Площадь квадрата.

Пример использования:

```python
# Пример 1
print(area(5))
# Вывод: 25

# Пример 2
print(perimeter(7,5))
# Вывод: 26.25
```

### perimeter(a)

Функция perimeter(a) вычисляет периметр квадрата с длиной стороны a.

Параметры:

- a (число): Длина стороны квадрата.

Возвращает:

- (число) Периметр квадрата.

Пример использования:

```python
# Пример 1
print(perimeter(4))
# Вывод: 16

# Пример 2
print(perimeter(7,5))
# Вывод: 26.25
```


## rectangle.py

представляет собой расчёт площади и периметра прямоугольника. Содержит две основные функции:

- `area(a, b)`: Вычисляет площадь прямоугольника по длинам его сторон.
- `perimeter(a, b)`: Вычисляет периметр прямоугольника по длинам его сторон.

#### Функция `area(a, b)`

** **
Вычисляет площадь прямоугольника по длинам его сторон.

**Параметры:**
- `a` (float): Длина первой стороны прямоугольника.
- `b` (float): Длина второй стороны прямоугольника.

**Возвращает:**
- `float`: Площадь прямоугольника.

### Пример использования:
```python
# Пример 1
side_a = 5
side_b = 3
print(area(side_a, side_b))
# Вывод: 15

# Пример 2
side_a = 10
side_b = 4
print(area(side_a, side_b))
# Вывод: 40
```

#### Функция `perimetr(a, b)`

** **
Вычисляет периметр прямоугольника по длинам его сторон.

**Параметры:**
- `a` (float): Длина первой стороны прямоугольника.
- `b` (float): Длина второй стороны прямоугольника.

**Возвращает:**
- `float`: Площадь прямоугольника.

### Пример использования:
```python
# Пример 1
side_a = 5
side_b = 3
print(area(side_a, side_b))
# Вывод: 16

# Пример 2
side_a = 8
side_b = 2
print(area(side_a, side_b))
# Вывод: 20
```

## triangle.py

Представляет собой расчёт площади и периметра треугольника

- `area(a, b)`: Вычисляет площадь треугольника по основанию и высоте.
- `perimeter(a, b, c)`: Вычисляет периметр прямоугольника по длинам его сторон.

#### Функция `area(a, b)`

** **
Вычисляет площадь прямоугольника по длинам его сторон.

**Параметры:**
- `a` (float): Длина первой стороны треугольника.
- `b` (float): Длина второй стороны треугольника.

**Возвращает:**
- `float`: Площадь треугольника.

### Пример использования:
```python
# Пример 1
side_a = 2
side_b = 5
print(perimeter(2.5))
# Вывод: 10

# Пример 2
side_a = 3
side_b = 10
print(perimeter(side_a, side_b))
# Вывод: 30
```

#### `perimeter(a, b, c)`

Функция для вычисления периметра треугольника.

**Параметры:**

- `a` (float): Длина первой стороны треугольника.
- `b` (float): Длина второй стороны треугольника.
- `c` (float): Длина третьей стороны треугольника.

**Возвращает:**

- `float`: Периметр треугольника.

**Пример использования:**

```python
# Пример 1
side_a = 3
side_b = 4
side_c = 5
print(perimeter(side_a, side_b, side_c))
# Вывод: 12

# Пример 2
side_a = 5.5
side_b = 6.1
side_c = 7.3
print(perimeter(side_a, side_b, side_c))
# Вывод: 18.9
```
