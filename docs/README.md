# Geometric_lib
This project is a collection of utility functions for performing various calculations.

## Installation
To install the project, simply clone the repository.

## Functions

### Perimiter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c

### Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle = a * h / 2
