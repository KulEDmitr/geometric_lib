Функция area(a)
Функция для вычисления площади квадрата.

Аргументы:

    a (float): Длина стороны квадрата.

Возвращаемое значение:

    float: Площадь квадрата.

Примеры вызова:

    result = area(7.77)
    print(result)  # Выведет: 60.3729
