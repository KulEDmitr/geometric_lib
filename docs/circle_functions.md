Функция area(r)
Функция для вычисления площади круга.

Аргументы:

    r (float): Радиус круга.

Возвращаемое значение:

float: Площадь круга.
Примеры вызова:

    result = area(3.0)
    print(result)  # Выведет: 28.274333882308138
